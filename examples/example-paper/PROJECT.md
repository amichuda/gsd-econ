# Mobile money and consumption smoothing in rural Uganda

## Research question

Does mobile-money agent rollout in rural districts allow households to smooth consumption against weather shocks via increased intra-household and inter-village remittance flow?

## Contribution

We provide the first quasi-experimental estimate of how mobile-money rollout affects consumption smoothing in rural Sub-Saharan Africa, leveraging staggered district-level rollout from 2008–2014 in Uganda. Prior work (Jack and Suri 2014 on Kenya M-Pesa) used cross-sectional variation; we use within-district variation in rollout timing and provide the first direct evidence on the remittance channel using matched sender-receiver mobile-money transaction records merged with household survey panels.
